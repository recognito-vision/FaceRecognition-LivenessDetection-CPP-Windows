1. Install Visual Studio 2019 or later
2. Install VulkanSDK from _dependency/VulkanSDK-1.4.304.0-Installer.exe
   Check Vulkan SDK Environment Variables:
	echo %VULKAN_SDK%
   This should output a valid Vulkan SDK path, such as:
	C:\VulkanSDK\1.4.304.0
3. Open and Run ttvfacedemo.sln project
4. Share HWID for license
5. ttvfacedemo.exe detects and compares face from "image1.jpg" and "image2.jpg" files.
